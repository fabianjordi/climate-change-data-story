## Gemacht mit

- [Flask](https://flask.palletsprojects.com) – Server
- [D3.js](https://d3js.org) – Visualisierungs-Library (eingesetzt in der Datenstory)
- [PostgreSQL](https://www.postgresql.org) – Datenbank
- [PonyORM](https://ponyorm.org) – ORM für PostgreSQL
- [Dash](https://dash.plot.ly/) - Dashboard für die interaktive Visualisierung
- [Heroku](https://www.heroku.com) – Hosting
